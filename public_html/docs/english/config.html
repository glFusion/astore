glFusion Astore Plugin Configuration

The Astore Plugin allows you to display Amazon products on your site

Main Astore Options

AWS Access Key
Enter your Amazon Associate Access Key here.
AWS Secret Key
Enter your Amazon Associate Secret Key here.
AWS Associate Tag
Enter your Amazon Associate Tag here.
AWS Region
Select the country code for the region associated with the country of your AWS associate tag.
Cache Lifetime in Minutes
Enter the length of time, in minutes, to cache data received from Amazon. To avoid exceeding request limits this value is adjusted up or down by 25% for each item. Zero indicates that caching is disabled, otherwise the value must be between 10 and 240.
AWS Debugging
Select "Yes" to log debugging messages related to Amazon queries.
Items to Show Per Page
Enter the number of items to show per page on the main store page.
Store Title
Enter the title for your store to appear on the main page.
Use Product API?
Select whether to use the Amazon Product API. If you enable the API you must fill in your keys below. If not, you will need to manually add the product link URLs for each product.
Add Featured Items to Catalog?
Select whether to automatically add featured items (those requested by item number) to the product catalog.
Maximum Featured Item Description Length
Enter the number of characters to be shown in the description for featured items.
Maximum Block Item Description Length
Enter the number of characters to be shown in the description for items displayed in blocks on the main page..
Block Associate ID for Header
Product links will not contain associate tag information if a HTTP header is set that matches this string. The value of the headeer is not used. This is to prevent artificially inflating click counts at Amazon Associate Central.
Block Associate ID for Admins
Product links will not contain associate tag information if this is "Yes" and the current site user is an administrator. This is to prevent artificially inflating click counts at Amazon Associate Central.
Enable Centerblock?
If the centerblock is enabled, the store will appear as the site's homepage.
Group allowed to search
Select the group which is allowed to search the catalog for items. Note that this will cause requests to be made to Amazon so you may wish to restrict this access to stay within the terms of service.
Default Cateogory
Select the category to which newly-found items will be added. You can change the category for individual items by editing the item record.
Items in catalog link to...
Select whether items shown in the catalog should link to the internal detail page or directly to Amazon.
